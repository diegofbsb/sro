Instalação do Aplicativo SRO no Coletor

10.1 Objetivo

A instalação do SRO.exe, aplicativo que iniciará o Sistema de Rastreamento de Objetos, ocorrerá em duas etapas:

  • A primeira etapa levará o aplicativo SRO.exe para o coletor. Este, por sua vez, se encarregará de instalar o aplicativo SRO.jar no coletor, bem como atualizá-lo cada vez que isto for processado no servidor SROserver (concentrador). Facilitando em muito a administração do sistema.

  • A Segunda etapa descreve o que está acontecendo no coletor quando o SRO.exe for acionado.

10.2 Primeira Etapa

Os Coletores Dolphin 7300 estão equipados com o Sistema Operacional Windows CE 3.0 e um dos aplicativos previamente instalados é o browser Internet Explorer.

Figura 55 - Conexão à rede de coletores

No Coletor, o IE está acessîvel pelo îcone no Desktop através do menu START > > Programs > Internet Explorer.  

Antes de realizar o acesso via browser, é necessário conectar-se à rede de coletores.

Importante

O Coletor deverá estar na doca para a conexão, pois é esta que realiza a comunicação entre a interface do coletor (infrared) e o cabeamento de rede (par trançado).
