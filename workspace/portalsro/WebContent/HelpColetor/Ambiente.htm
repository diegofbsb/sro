1.3 Ambiente

O ambiente é dividido em duas partes:

  • Rede Corporativa, que corresponde aos equipamentos já existentes e onde serão instalados os aplicativos com sua devida parametrização;

  • Rede de Coletores, que dispõe dos componentes a serem instalados e parametrizados de acordo com esta documentação.

 

1.4 Componentes

Os componentes devem ser entendidos como parte da solução.

1.4.1 Rede Corporativa

Hardware

  • Concentrador - servidor onde serão instalados os aplicativos SROServer e de ambiente.

Software

  • Windows NT 4.0 SP6a;

  • Java Run Time 1.4.2.03;

  • MDAC 2.5 (recomenda-se fortemente utilizar somente esta versão);

  • Driver para Impressora Intermec EasyCode 3400 (opcional);

  • SROserver.

Os softwares para instalação nos coletores estarão disponîveis no Concentrador.

 

1.4.2 Rede de Coletores

Hardware

  • Coletor de Dados HHP modelo Dolphin 7300;

  • Net Base HHP;

  • Roteador;

  • Impressora de Relatórios modelo Epson C43SX.

Software

  • Aplicativo SRO
