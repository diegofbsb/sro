5.5 Parâmetros para os Coletores (PARA)

Os parâmetros de configuração do SRO, utilizados pelo SRO.JAR são lidos de um arquivo denominado PARA.

Durante a instalação do SRO.JAR no coletor, o arquivo PARA é copiado do servidor para coletor.

Sendo assim, antes de iniciar a instalação dos coletores em uma unidade, deve-se editar o arquivo PARA no servidor e ajustar os parâmetros de acordo com esta unidade.

A edição do arquivo PARA no servidor é feita através do software EDITPARA que é chamado através do bat EDITPARA.BAT, disponível na pasta \SROSERVER.
