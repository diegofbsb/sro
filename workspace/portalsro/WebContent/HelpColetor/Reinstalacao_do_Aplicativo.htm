10.4 Reinstalação do Aplicativo

Caso seja necessário, o aplicativo SRO poderá ser reinstalado.

Caso o coletor não inicie a instalação automaticamente, é possîvel comandar (forçar) a

instalação pressionando-se as teclas ALT+I.

Este conjunto de teclas (ALT+I) somente estará disponîvel na última tela de login do Aplicativo SRO, que é apresentada sempre que o coletor já possuir uma instalação do SRO.jar.

Figura 77 - SRO - Tela de login

Ao pressionar Alt+I nesta tela, é solicitado que seja preenchido o endereço IP.

Figura 78 - Informando endereço IP

Ao informar este endereço, o coletor deverá ser docado para que os arquivos sejam transferidos do servidor para o coletor. Também nesta etapa certifique-se de que o servidor está rodando normalmente.
