Uso dos filtros:

  • Caso haja uma caixa de seleção, apenas escolha um valor
  • Se for uma caixa de texto, digite parte da palavra que deseja buscar.
  • Se o conteúdo for apenas numérico, pode-se usar os sinais "<", "<=", ">", ">=".
  • Para selecionar os intervalos de tempo maiores que um dia (coluna 'Tempo'), copie e cole no campo apropriado a seguinte expressão: 00A([0-9][1-9]|[1-9][0-9])D
  • Para selecionar os intervalos de tempo menores que um dia (coluna 'Tempo'), copie e cole no campo apropriado a seguinte expressão: 00A00D
  • Use '^' seguido de uma expressão para selecionar todas as palavras que comecem com tal expressão. Exemplo: ^ACF
  • Pressione enter para aplicar os filtros após digitá-los em uma caixa de texto
  • Clique em "Reset" para limpar os filtros.
